A Simple, Responsive CSS Grid System

Published on March 13, 2014.

We are currently building a new CMS at Big Room Studios, and I was given the task of writing a fluid CSS grid system for said CMS.

Although this CSS grid is fluid, it also had to be somewhat rigid to start. We didn’t want to give users too much control in the MVP stage.

Because of this, all grid columns simply stack on top of each other on smaller screens, as 100% width blocks. It’s nothing revolutionary, but the code is super minimal and I wanted to share it here in case anyone else found it useful.
